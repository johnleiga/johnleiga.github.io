Project Summary

  • Project Type: Aerospace hardware redesign
  • Role: Project Engineer
  • Primary Focus: Architecture redesign, cost reduction, validation
  • Outcome: Preserved a $40M+ program
  • Key Results: ~30% system cost reduction, ~50% gage component cost reduction

Redesigning the architecture instead of trimming the edges

This project focused on a high-reliability aerospace pressure transducer used in launch vehicle propulsion, satellite propellant tanks, and cryogenic service.

A major customer required a steep cost reduction that put a $40M+ program at risk. The existing design did not have enough margin to absorb that reduction through incremental changes, so I approached the problem at the architecture level.

What I did

I identified the source of reference pressure error and redesigned the diaphragm and gage geometry so the structure operated primarily in compression instead of expansion. That change reduced the error enough to eliminate the secondary reference pressure gage.

Removing that subsystem also enabled the use of lower-cost standard electronics while maintaining performance requirements.

I completed full assembly design in SolidWorks, performed structural analysis, created GD&T drawing packages, and worked through tolerance considerations around diaphragm motion and sensor gap. I then built and validated prototype hardware across critical pressure ranges to verify that the redesign met requirements and eliminated the original error source.

Why it mattered

  • Preserved a major aerospace program that was at risk due to cost pressure
  • Achieved approximately 30% system cost reduction
  • Reduced mechanical gage component cost by approximately 50%
  • Validated the redesign through prototype build and testing

Engineering themes

This project reflects first-principles reasoning, architecture-level design, and prototype validation used to improve both performance and cost.
