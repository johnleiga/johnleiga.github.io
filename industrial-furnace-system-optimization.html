Project Summary

  • Project Type: Industrial manufacturing optimization
  • Environment: 14-furnace live production system
  • Primary Focus: Process stability, bottleneck removal, startup automation
  • Outcome: Stabilized production and eliminated startup waste
  • Key Results: ~$200K annual savings and ~$500K in executed capital upgrades

Production engineering in a live operating environment

This project focused on a 14-furnace perlite expansion system that was unstable, labor-intensive, and wasteful at startup.

Because the work was done in a live factory environment, the solution had to improve performance without sacrificing operability or reliability.

Mechanical and controls improvements that held up in production

I approached the system as an integrated process rather than a collection of isolated issues. I installed sample points, collected operating data, and used that data to identify the primary drivers of instability and waste.

I stabilized performance across the furnace system by tuning interacting control loops in the correct sequence and improving overall process behavior. I also identified an undersized rotary airlock as a shutdown-causing bottleneck and helped eliminate that constraint.

To improve product consistency, I implemented density-based feedback control that reduced operator intervention and improved control of the finished product.

I also developed and deployed a zero-waste automated startup sequence. That work required test runs, parameter development, and controls changes that eliminated startup material loss, saved approximately $200K annually, and scaled across facilities in Bloomfield, MO and King William, VA.

The broader effort also supported approximately $500K in feed system upgrades, including equipment planning, vendor coordination, installation, and commissioning.

Why it mattered

  • Stabilized a live 14-furnace production system
  • Removed a shutdown-causing bottleneck
  • Reduced startup waste and saved approximately $200K annually
  • Scaled a proven startup improvement across multiple facilities

Engineering themes

This project reflects root-cause analysis, controls integration, process optimization, and implementation in live production systems where solutions must work reliably in operation.
