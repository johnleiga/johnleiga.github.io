Aerospace Pressure Transducer Redesign

  1. Home
  2. Transducer Project

Project Summary

  • Project Type: Aerospace hardware redesign
  • Role: Project Engineer
  • Primary Focus: Mechanical architecture, cost reduction, validation
  • Outcome: Preserved a $40M+ program
  • Key Results: ~30% system cost reduction, ~50% gage component cost reduction

Redesigning the architecture instead of trimming the edges

This project centered on a high-reliability aerospace pressure transducer used in demanding environments such as launch vehicle propulsion systems, satellite propellant tanks, and cryogenic applications.

A major customer required an aggressive cost reduction target that threatened the competitiveness of a $40M+ program. Rather than treating the problem as a simple cost-cutting exercise, I approached it as a structural and sensing problem.

What I did

I analyzed the root cause of reference pressure error in the existing transducer architecture and redesigned the diaphragm and gage geometry so the structure operated primarily in compression rather than expansion. That change reduced the underlying error mechanism enough to eliminate the secondary reference pressure gage.

The redesign also enabled a shift to lower-cost standard electronics, simplifying the architecture while maintaining performance requirements. My work included full assembly design in SolidWorks, structural analysis, GD&T drawing packages, tolerance considerations affecting diaphragm motion and sensor gap, and prototype validation at the most challenging pressure ranges.

Why it mattered

  • Preserved competitiveness on a major aerospace program
  • Achieved approximately 30% system cost reduction
  • Reduced mechanical gage component cost by approximately 50%
  • Validated the design through prototype build and testing

Engineering themes

This project reflects the kind of work I find most valuable: first-principles reasoning, architecture-level design changes, and prototype-backed decisions that improve both technical performance and commercial viability.
